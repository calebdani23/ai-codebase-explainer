# Demo Script

1. Open the landing page and explain the product goal.
2. Click **Try Demo Repository**.
3. Show the repository intake form and security note.
4. Run demo analysis and open the analysis overview.
5. Show suggested issues with severity, priority, confidence, effort and related files.
6. Open `/api/analyses/<analysis_id>/export.md` to show the portfolio-quality Markdown report.
7. Explain that OpenAI issue generation is optional and backend-only; without a key, deterministic heuristics still produce actionable issues.
8. Open the observability section and explain that analysis traces can be sent to the prior AI Agent Observability Dashboard.
